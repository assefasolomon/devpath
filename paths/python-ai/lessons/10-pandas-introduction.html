Pandas is the most widely used data manipulation library in Python. While NumPy works with raw numerical arrays, Pandas works with labelled, structured data — the kind you find in CSV files, databases, and spreadsheets. Before any AI model can be trained, data must be loaded, inspected, and cleaned. Pandas is the tool that makes this possible.

Lesson Overview

  • Create DataFrames from dictionaries, lists, and CSV files.
  • Inspect DataFrames with head(), info(), describe(), and shape.
  • Select columns and rows using [], .loc[], and .iloc[].

The Series

A Series is a one-dimensional labelled array — like a single column in a spreadsheet. Each value has an index label.

import pandas as pd

# Create a Series from a list
scores = pd.Series([0.85, 0.92, 0.78, 0.95, 0.88])
print(scores)
# 0    0.85
# 1    0.92
# 2    0.78
# 3    0.95
# 4    0.88

# Custom index labels
scores = pd.Series(
    [0.85, 0.92, 0.78, 0.95],
    index=["Alice", "Bob", "Carol", "David"]
)
print(scores["Alice"])   # 0.85
print(scores.mean())     # 0.875
print(scores.max())      # 0.95

The DataFrame

A DataFrame is a two-dimensional labelled data structure — like a spreadsheet or SQL table. Each column is a Series. This is the primary object you will work with in every AI data pipeline.

import pandas as pd

# Create from a dictionary — keys become column names
data = {
    "name":     ["Alice", "Bob", "Carol", "David", "Eve"],
    "age":      [24, 31, 28, 35, 22],
    "score":    [0.85, 0.92, 0.78, 0.95, 0.88],
    "passed":   [True, True, True, True, True]
}
df = pd.DataFrame(data)
print(df)
#     name  age  score  passed
# 0  Alice   24   0.85    True
# 1    Bob   31   0.92    True
# 2  Carol   28   0.78    True
# 3  David   35   0.95    True
# 4    Eve   22   0.88    True

Inspecting a DataFrame

Always inspect your data before doing anything else. These methods give you a complete picture of what you are working with.

import pandas as pd

df = pd.read_csv("students.csv")  # load real data

print(df.shape)        # (rows, columns) e.g. (150, 5)
print(df.head())       # first 5 rows
print(df.head(10))     # first 10 rows
print(df.tail())       # last 5 rows
print(df.columns)      # column names
print(df.dtypes)       # data type of each column
print(df.info())       # summary: dtypes + non-null counts
print(df.describe())   # stats: mean, std, min, max etc.
print(df.isnull().sum())  # count missing values per column
💡 Always run df.info() and df.describe() the moment you load a new dataset. These two commands tell you everything about what needs to be cleaned before you can use the data in a model.

Selecting Columns

import pandas as pd

data = {
    "name":  ["Alice", "Bob", "Carol", "David", "Eve"],
    "age":   [24, 31, 28, 35, 22],
    "score": [0.85, 0.92, 0.78, 0.95, 0.88],
    "grade": ["B", "A", "C", "A", "B"]
}
df = pd.DataFrame(data)

# Single column — returns a Series
print(df["score"])

# Multiple columns — returns a DataFrame
print(df[["name", "score"]])

# Column as attribute (only works for simple names)
print(df.score)     # same as df["score"]

# Add a new column
df["score_pct"] = df["score"] * 100
print(df.head())

Selecting Rows: loc and iloc

.loc[] selects by label (index name). .iloc[] selects by integer position. Both work for rows and columns.

import pandas as pd

data = {
    "name":  ["Alice", "Bob", "Carol", "David", "Eve"],
    "age":   [24, 31, 28, 35, 22],
    "score": [0.85, 0.92, 0.78, 0.95, 0.88]
}
df = pd.DataFrame(data)

# iloc — by integer position (like NumPy)
print(df.iloc[0])          # first row
print(df.iloc[0, 1])       # row 0, column 1 → 24
print(df.iloc[1:3])        # rows 1 and 2
print(df.iloc[:, 1:])      # all rows, columns 1 onwards

# loc — by label
print(df.loc[0, "name"])   # 'Alice'
print(df.loc[2:4, ["name", "score"]])  # rows 2-4, two columns

# Set a custom index
df = df.set_index("name")
print(df.loc["Bob"])       # row where name == 'Bob'
print(df.loc["Bob", "score"])  # 0.92

Filtering Rows

Boolean filtering in Pandas works exactly like NumPy boolean masking — apply a condition to get a boolean Series, then use it to select rows.

import pandas as pd

data = {
    "name":  ["Alice", "Bob", "Carol", "David", "Eve"],
    "age":   [24, 31, 28, 35, 22],
    "score": [0.85, 0.92, 0.78, 0.95, 0.88],
    "grade": ["B", "A", "C", "A", "B"]
}
df = pd.DataFrame(data)

# Single condition
high = df[df["score"] >= 0.90]
print(high)

# Multiple conditions — use & and |
top = df[(df["score"] >= 0.85) & (df["age"] < 30)]
print(top)

# Filter by string value
grade_a = df[df["grade"] == "A"]
print(grade_a)

# isin — match multiple values
selected = df[df["grade"].isin(["A", "B"])]
print(selected)

Reading and Writing CSV Files

In real AI projects you almost always load data from files rather than creating DataFrames by hand.

import pandas as pd

# Read a CSV file
df = pd.read_csv("data/students.csv")

# Read with specific options
df = pd.read_csv(
    "data/students.csv",
    index_col=0,          # use first column as index
    parse_dates=["date"], # parse date columns
    na_values=["N/A", "?"] # treat these as NaN
)

# Write back to CSV
df.to_csv("data/students_clean.csv", index=False)

# Read Excel
df = pd.read_excel("data/report.xlsx", sheet_name="Sheet1")

# Read JSON
df = pd.read_json("data/records.json")
