It's time to put Unit 1 into practice. You will build a command-line tool that accepts a list of numbers from the user and computes a full statistical summary. This project uses only pure Python — no NumPy yet — so you cement your understanding of data types, loops, functions, and modules before relying on libraries to do the work.

Project Overview
  • Build a CLI tool that takes numeric input from the user in a loop.
  • Compute mean, median, mode, variance, and standard deviation using pure Python.
  • Organise your code into a stats.py module and a main.py entry point.
  • Handle edge cases: empty input, non-numeric input, division by zero.

Requirements

  • Prompt the user to enter numbers one at a time, stopping when they type done.
  • Reject non-numeric input gracefully — no crashes.
  • Compute and display: count, sum, minimum, maximum, mean, median, mode, variance, and standard deviation.
  • All statistical functions must live in a separate stats.py module with docstrings.
  • main.py handles user interaction only — no stat logic lives there.

# main.py
from stats import mean, median, mode, variance, std_dev

def collect_numbers():
    """Prompt the user and return a list of floats."""
    numbers = []
    print("\nData Calculator — enter numbers one at a time.")
    print("Type 'done' when finished.\n")
    while True:
        entry = input("Enter a number: ").strip()
        if entry.lower() == "done":
            break
        try:
            numbers.append(float(entry))
        except ValueError:
            print("  Invalid input. Please enter a number.")
    return numbers

def display_results(numbers):
    """Print a statistical summary."""
    if not numbers:
        print("No numbers entered.")
        return
    print("\n--- Results ---")
    print(f"{'Count':<12}: {len(numbers)}")
    print(f"{'Sum':<12}: {sum(numbers)}")
    print(f"{'Minimum':<12}: {min(numbers)}")
    print(f"{'Maximum':<12}: {max(numbers)}")
    print(f"{'Mean':<12}: {mean(numbers):.1f}")
    print(f"{'Median':<12}: {median(numbers):.1f}")
    print(f"{'Mode':<12}: {mode(numbers):.1f}")
    print(f"{'Variance':<12}: {variance(numbers):.1f}")
    print(f"{'Std Dev':<12}: {std_dev(numbers):.2f}")

if __name__ == "__main__":
    data = collect_numbers()
    display_results(data)

Hints

  • Median: Sort with sorted(numbers). If length is even, average the two middle values.
  • Mode: Build a frequency dict with a loop, then use max(freq, key=freq.get).
  • Variance: sum((x - m)**2 for x in numbers) / len(numbers) where m is the mean.
  • Std Dev: import math then math.sqrt(variance(numbers)).
💡 Always use if __name__ == "__main__": at the bottom of main.py. This ensures the script only runs when executed directly, not when imported as a module.
