This project puts everything from Unit 2 into practice. You will build a NumPy-powered toolkit that implements the core mathematical operations behind a single neural network forward pass — without using any ML framework. By the end, you will have implemented data normalisation, a linear layer, three activation functions, and a loss function entirely from scratch using NumPy.

Project Overview
  • Normalise a dataset using min-max and z-score normalisation.
  • Implement a linear (fully connected) layer using matrix multiplication.
  • Implement ReLU, Sigmoid, and Softmax activation functions.
  • Compute Mean Squared Error (MSE) and Cross-Entropy loss.
  • Organise all functions into a nn_utils.py module.

Requirements

  • All functions must live in nn_utils.py with docstrings.
  • main.py imports from nn_utils.py and demonstrates each function with printed output.
  • Use only NumPy — no Scikit-learn, TensorFlow, or PyTorch.
  • All functions must handle arbitrary input shapes correctly.

from nn_utils import (minmax_normalise, zscore_normalise,

np.random.seed(42)

data = np.array([10, 25, 5, 40, 15], dtype=float)

print(f"Z-Score:       {zscore_normalise(data).round(2)}")

print("\n--- Linear Layer ---")
X = np.random.randn(4, 3)     # 4 samples, 3 features
W = np.random.randn(3, 2)     # 3 inputs, 2 outputs
b = np.zeros(2)
out = linear_layer(X, W, b)
print(f"Input shape:   {X.shape}")
print(f"Weights shape: {W.shape}")
print(f"Output shape:  {out.shape}")
print(f"Output:\n{out.round(2)}")

print("\n--- Activation Functions ---")
x = np.array([-2.0, -1.0, 0.0, 0.5, 1.2, 2.3])
print(f"ReLU:     {relu(x)}")
print(f"Sigmoid:  {sigmoid(x).round(3)}")
logits = np.array([1.0, 2.0, 3.5, 4.1])
probs = softmax(logits)
print(f"Softmax:  {probs.round(3)} — sums to {probs.sum():.1f}")

print("\n--- Loss Functions ---")
y_true = np.array([1.0, 0.0, 1.0, 1.0])
y_pred = np.array([0.9, 0.2, 0.8, 0.7])
print(f"MSE Loss:          {mse_loss(y_true, y_pred):.4f}")

y_true_ce = np.array([0, 0, 1, 0])   # one-hot
y_pred_ce = softmax(np.array([0.5, 1.0, 2.5, 0.3]))
print(f"Cross-Entropy:     {cross_entropy_loss(y_true_ce, y_pred_ce):.4f}")

Hints

  • ReLU: np.maximum(0, X) is the cleanest one-liner.
  • Sigmoid: 1 / (1 + np.exp(-X)).
  • Softmax stability: subtract X.max() before applying np.exp() to prevent overflow.
  • Cross-entropy: use np.clip(y_pred, 1e-15, 1 - 1e-15) before np.log() to avoid log(0) = -inf.
  • Linear layer bias: check if b is not None before adding it.
💡 These exact functions — ReLU, sigmoid, softmax, cross-entropy — are what TensorFlow and PyTorch implement under the hood. You are not just learning NumPy; you are learning how deep learning actually works.
