Control flow is how a program makes decisions and repeats work. In AI and data pipelines, conditionals decide how to handle missing values or outliers, and loops process every row in a dataset or every epoch of training. Mastering these patterns is essential before working with any data library.

Lesson Overview
  • Use if, elif, and else to branch logic.
  • Use for loops to iterate over sequences and ranges.
  • Use while loops for condition-based repetition.
  • Control loop execution with break, continue, and pass.
  • Write concise list comprehensions.

Conditionals: if / elif / else

Python uses indentation (4 spaces) to define blocks — there are no curly braces. The if statement evaluates a condition and runs its block only when the condition is True.

accuracy = 0.87

if accuracy >= 0.95:
    print("Excellent — deploy to production")
elif accuracy >= 0.80:
    print("Good — continue tuning")
elif accuracy >= 0.60:
    print("Mediocre — review architecture")
else:
    print("Poor — check data quality")

# Output: Good — continue tuning
💡 Python has a one-line ternary expression: label = "pass" if score >= 0.5 else "fail". Use it for simple assignments, but prefer full if/else blocks for complex logic.

for Loops

The for loop iterates over any sequence — a list, string, dictionary, or range. It is the workhorse of data processing.

# Iterate over a list
losses = [1.42, 1.10, 0.83, 0.61, 0.45]
for loss in losses:
    print(f"Loss: {loss:.2f}")

# range(start, stop, step)
for epoch in range(1, 6):
    print(f"Epoch {epoch}")

# enumerate — gives index AND value
labels = ["cat", "dog", "bird"]
for i, label in enumerate(labels):
    print(f"{i}: {label}")

# Iterate over a dictionary
config = {"lr": 0.001, "batch": 32, "epochs": 50}
for key, value in config.items():
    print(f"{key} = {value}")

while Loops

The while loop runs as long as its condition stays True. In AI it is useful for training loops that run until a loss threshold is reached.

loss = 2.0
epoch = 0

while loss > 0.5:
    loss *= 0.75
    epoch += 1
    print(f"Epoch {epoch}: loss = {loss:.4f}")

print(f"Converged after {epoch} epochs")
⚠️ Always ensure a while loop's condition will eventually become False, or add a maximum iteration counter — otherwise your program will loop forever.

break, continue, and pass

accuracies = [0.72, 0.81, 0.95, 0.60, 0.88]

# break — exit the loop immediately
for acc in accuracies:
    if acc >= 0.95:
        print(f"Target reached: {acc}")
        break

# continue — skip to the next iteration
for acc in accuracies:
    if acc < 0.80:
        continue
    print(f"Accepted: {acc}")

# pass — placeholder for empty blocks
for acc in accuracies:
    if acc < 0.70:
        pass
    else:
        print(f"OK: {acc}")

List Comprehensions

List comprehensions are a concise, Pythonic way to build a new list by transforming or filtering an existing sequence. They replace many simple for loops in one readable line.

raw_scores = [0.92, 0.45, 0.87, 0.30, 0.95, 0.61]

# Transform: multiply each score by 100
percentages = [score * 100 for score in raw_scores]
# [92.0, 45.0, 87.0, 30.0, 95.0, 61.0]

# Filter: keep only passing scores
passing = [s for s in raw_scores if s >= 0.60]
# [0.92, 0.87, 0.95, 0.61]

# Transform + filter combined
high_pct = [round(s * 100, 1) for s in raw_scores if s >= 0.80]
# [92.0, 87.0, 95.0]

# Dictionary comprehension
labels = ["cat", "dog", "bird"]
label_to_id = {label: i for i, label in enumerate(labels)}
# {'cat': 0, 'dog': 1, 'bird': 2}
