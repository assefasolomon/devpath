Python has become the dominant language of Artificial Intelligence and Data Science — not by accident, but because its readable syntax, massive ecosystem, and interactive tools make it ideal for exploring data, building models, and communicating results. This lesson explains why Python is the right choice for AI work and gets your environment ready to write real code.

Lesson Overview
  • Understand why Python is the primary language for AI and Data Science.
  • Install Python and set up a working development environment.
  • Write and run your first Python script using the interactive interpreter.
  • Understand what pip is and how to install AI libraries.
  • Get familiar with Jupyter Notebooks.

Why Python for AI?

Python is used by over 85% of data scientists as their primary tool. Several factors make it the go-to language for AI:

  • Readable syntax — Python reads almost like English, so you focus on the problem, not the language.
  • Rich ecosystem — NumPy, Pandas, Matplotlib, Scikit-learn, TensorFlow, and PyTorch are all Python-first.
  • Interactive development — Jupyter Notebooks let you write code, see results, and document thinking in one place.
  • Community — The largest open-source AI community means fast answers and continuous library updates.
ℹ️ Every major AI framework — TensorFlow, PyTorch, Scikit-learn, Hugging Face Transformers — has Python as its primary interface. Learning Python is learning the language of modern AI.

Installing Python

The recommended approach for AI work is to install Python through Anaconda, a distribution that bundles Python with the most important data science libraries.

  1. Download Anaconda from anaconda.com/download.
  2. Run the installer and accept default settings.
  3. Open Anaconda Navigator and launch Jupyter Notebook or VS Code.
💡 Prefer a lighter setup? Install Python directly from python.org and use pip to install libraries individually.

Your First Python Script

Open a terminal and type python3 to enter the interactive interpreter.

>>> print("Hello, AI world!")
Hello, AI world!

>>> 2 + 2
4

>>> "Python" * 3
'PythonPythonPython'

To write a saved script, create a file hello.py and run it:

# hello.py
name = "AI Engineer"
print(f"Hello, {name}!")

import sys
print("Python version:", sys.version)
$ python3 hello.py
Hello, AI Engineer!
Python version: 3.11.5

Installing Libraries with pip

# Install the core AI/Data Science stack
pip install numpy pandas matplotlib

# Verify
pip show numpy

Jupyter Notebooks

Jupyter Notebooks are the standard tool for AI and data work. Code, output, and explanation all live in the same document.

# Launch from terminal
jupyter notebook
# or
jupyter lab

A browser tab opens at http://localhost:8888. Press Shift + Enter to run a cell.
