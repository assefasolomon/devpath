Real-world data is messy. Missing values, wrong data types, duplicate rows, inconsistent formatting — these problems exist in every dataset you will ever work with. Data scientists spend up to 80% of their time cleaning data before any modelling begins. This lesson teaches you the Pandas tools to handle every common cleaning scenario.

Lesson Overview
  • Detect and handle missing values with isnull(), dropna(), and fillna().
  • Remove duplicate rows with drop_duplicates().
  • Rename columns and change data types with astype().
  • Add new columns and transform existing ones.
  • Apply custom functions to columns with apply() and map().

Handling Missing Values

Missing values appear as NaN (Not a Number) in Pandas. Always check for them as the first step of any data cleaning pipeline.

import pandas as pd
import numpy as np

df = pd.DataFrame({
    "name":  ["Alice", "Bob", None, "David", "Eve"],
    "age":   [24, None, 28, 45, 35],
    "score": [0.92, 0.85, 0.78, None, 0.88]
})

# Detect missing values
print(df.isnull())          # True where value is missing
print(df.isnull().sum())    # count of NaN per column
print(df.isnull().sum().sum())  # total missing values

# Drop rows with ANY missing value
clean = df.dropna()
print(clean.shape)  # (3, 3)

# Drop rows only if ALL values are missing
df.dropna(how="all")

# Drop columns with missing values
df.dropna(axis=1)

# Fill missing values
df["age"].fillna(df["age"].mean(), inplace=True)    # with mean
df["score"].fillna(df["score"].median(), inplace=True)  # with median
df["name"].fillna("Unknown", inplace=True)           # with constant

print(df.isnull().sum())  # should all be 0 now
💡 Use inplace=True to modify the DataFrame directly, or assign the result: df = df.fillna(0). Both are equivalent — pick one style and stick to it.

Removing Duplicates

df = pd.DataFrame({
    "name":  ["Alice", "Bob", "Alice", "Carol", "Bob"],
    "score": [0.92, 0.85, 0.92, 0.78, 0.85]
})

print(df.duplicated())          # True for duplicate rows
print(df.duplicated().sum())    # 2 duplicates

# Remove duplicate rows — keep first occurrence
df = df.drop_duplicates()
print(df.shape)  # (3, 2)

# Remove duplicates based on specific column
df.drop_duplicates(subset=["name"], keep="last")

Renaming and Changing Data Types

import pandas as pd

df = pd.DataFrame({
    "Name": ["Alice", "Bob", "Carol"],
    "AGE": ["24", "32", "28"],       # stored as strings — wrong
    "Score": [0.92, 0.85, 0.78]
})

# Rename columns
df = df.rename(columns={"Name": "name", "AGE": "age", "Score": "score"})
print(df.columns)  # ['name', 'age', 'score']

# Change data type
df["age"] = df["age"].astype(int)
print(df.dtypes)
# name      object
# age        int64
# score    float64

# Convert to category — saves memory for low-cardinality columns
df["name"] = df["name"].astype("category")

Adding and Transforming Columns

import pandas as pd

df = pd.DataFrame({
    "name":  ["Alice", "Bob", "Carol", "David"],
    "score": [0.92, 0.85, 0.78, 0.95]
})

# Add a new column
df["percentage"] = df["score"] * 100
df["passed"]     = df["score"] >= 0.80

# Add a grade column using a function
def assign_grade(score):
    if score >= 0.90: return "A"
    elif score >= 0.80: return "B"
    elif score >= 0.70: return "C"
    else: return "F"

df["grade"] = df["score"].apply(assign_grade)
print(df)
#     name  score  percentage  passed grade
# 0  Alice   0.92        92.0    True     A
# 1    Bob   0.85        85.0    True     B
# 2  Carol   0.78        78.0   False     C
# 3  David   0.95        95.0    True     A

apply() and map()

apply() runs a function on each element or row of a DataFrame. map() is used on a Series to substitute values.

import pandas as pd

df = pd.DataFrame({
    "name":  ["alice", "BOB", "Carol"],
    "score": [0.92, 0.85, 0.78]
})

# apply() on a column — normalise names
df["name"] = df["name"].apply(str.title)
print(df["name"])  # ['Alice', 'Bob', 'Carol']

# apply() with lambda
df["score_pct"] = df["score"].apply(lambda x: round(x * 100, 1))

# map() — substitute values using a dictionary
grade_map = {"Alice": "senior", "Bob": "junior", "Carol": "junior"}
df["level"] = df["name"].map(grade_map)
print(df)

String Operations

Pandas provides vectorised string methods via .str — no loops needed for cleaning text columns.

import pandas as pd

df = pd.DataFrame({
    "email": ["  Alice@Gmail.COM ", "bob@yahoo.com", "CAROL@hotmail.com  "]
})

# Clean email column
df["email"] = (df["email"]
    .str.strip()        # remove leading/trailing spaces
    .str.lower()        # convert to lowercase
)
print(df["email"])
# ['alice@gmail.com', 'bob@yahoo.com', 'carol@hotmail.com']

# Extract domain
df["domain"] = df["email"].str.split("@").str[1]
print(df["domain"])  # ['gmail.com', 'yahoo.com', 'hotmail.com']

# Check if contains a substring
print(df["email"].str.contains("gmail"))  # [True, False, False]
