Functions are reusable blocks of code that take inputs and return outputs. In AI workflows, you wrap every repeated operation — data cleaning, feature engineering, metric calculation — into a function. Modules and packages let you organise those functions and tap into Python's vast library ecosystem.

Lesson Overview
  • Define functions with def, parameters, default values, and return statements.
  • Understand variable scope: local vs global.
  • Write lambda functions for short one-line operations.
  • Import standard library modules and third-party packages.
  • Organise your own code into modules.

Defining Functions

Use the def keyword to define a function. A function can accept parameters, compute something, and return a result. Good AI code wraps each pipeline stage into its own function.

def calculate_accuracy(correct, total):
    """Return accuracy as a float between 0 and 1."""
    if total == 0:
        return 0.0
    return correct / total

result = calculate_accuracy(87, 100)
print(result)          # 0.87
print(f"{result:.1%}") # 87.0%

Default Arguments and *args / **kwargs

# Default argument — used when caller omits it
def train_model(epochs=10, learning_rate=0.001, verbose=True):
    for epoch in range(1, epochs + 1):
        if verbose:
            print(f"Epoch {epoch}/{epochs} — lr={learning_rate}")

train_model()               # uses all defaults
train_model(epochs=5)       # override just epochs
train_model(50, 0.0001, False)

# *args — variable number of positional arguments
def average(*scores):
    return sum(scores) / len(scores)

print(average(0.9, 0.85, 0.92))  # 0.89

# **kwargs — variable number of keyword arguments
def log_metrics(**metrics):
    for name, value in metrics.items():
        print(f"  {name}: {value:.4f}")

log_metrics(loss=0.231, accuracy=0.912, f1=0.905)

Return Values

Functions can return multiple values as a tuple — useful for returning a result alongside metadata.

def evaluate_model(predictions, labels):
    """Return accuracy and number of errors."""
    correct = sum(p == l for p, l in zip(predictions, labels))
    total = len(labels)
    accuracy = correct / total
    errors = total - correct
    return accuracy, errors

preds  = [1, 0, 1, 1, 0]
actual = [1, 0, 0, 1, 1]

acc, errs = evaluate_model(preds, actual)
print(f"Accuracy: {acc:.0%}, Errors: {errs}")
# Accuracy: 60%, Errors: 2

Variable Scope

Variables defined inside a function are local — they only exist within that function. Variables defined outside are global.

threshold = 0.80  # global variable

def check_accuracy(acc):
    result = "pass" if acc >= threshold else "fail"  # reads global
    return result

print(check_accuracy(0.92))  # pass
print(check_accuracy(0.71))  # fail

# result is not accessible here — it is local to check_accuracy
⚠️ Avoid modifying global variables inside functions — it makes code hard to debug. Pass values as parameters and return results instead.

Lambda Functions

A lambda is a small anonymous function defined in a single expression. Commonly used with sorted(), map(), and filter().

normalize = lambda x, max_val: x / max_val
print(normalize(128, 255))  # 0.502

# Sort models by accuracy descending
models = [
    {"name": "ModelA", "accuracy": 0.87},
    {"name": "ModelB", "accuracy": 0.92},
    {"name": "ModelC", "accuracy": 0.79},
]
ranked = sorted(models, key=lambda m: m["accuracy"], reverse=True)
for m in ranked:
    print(f"{m['name']}: {m['accuracy']}")

Importing Modules

A module is any Python file. Python's standard library includes hundreds of useful modules. Third-party packages like NumPy and Pandas are imported the same way after installing with pip.

# Import whole module
import math
print(math.sqrt(144))  # 12.0
print(math.pi)         # 3.14159...

# Import specific names
from math import log, exp
print(log(10))         # 2.302...

# Standard aliases used in every AI project
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

Creating Your Own Module

Any .py file is a module. Create metrics.py in the same directory and import from it in any other file.

# metrics.py
def accuracy(correct, total):
    """Return accuracy as a float."""
    return correct / total if total else 0.0

def precision(true_positives, false_positives):
    """Return precision score."""
    denom = true_positives + false_positives
    return true_positives / denom if denom else 0.0
# main.py — import and use your module
from metrics import accuracy, precision

print(accuracy(90, 100))   # 0.9
print(precision(45, 5))    # 0.9
