Choosing the right chart type is just as important as the data itself. A line chart communicates trends over time. A bar chart compares categories. A scatter plot reveals relationships between two variables. A histogram shows how data is distributed. This lesson teaches you when and how to use each of these four essential chart types in AI and data science work.

Lesson Overview
  • Create line charts for trends and time series data.
  • Create bar charts and horizontal bar charts for category comparisons.
  • Create scatter plots for correlation and cluster visualisation.
  • Create histograms for distribution analysis.
  • Know when to use each chart type.

Line Chart — Trends Over Time

Use a line chart when your x-axis represents time or a continuous sequence and you want to show how a value changes. In AI, line charts are used to visualise training loss and accuracy over epochs.

import matplotlib.pyplot as plt
import numpy as np

epochs = np.arange(1, 21)
train_loss = 2.5 * np.exp(-0.2 * epochs) + np.random.normal(0, 0.05, 20)
val_loss   = 2.5 * np.exp(-0.18 * epochs) + np.random.normal(0, 0.08, 20)

fig, ax = plt.subplots(figsize=(9, 4))

ax.plot(epochs, train_loss, color="steelblue",
        linewidth=2, label="Training Loss")
ax.plot(epochs, val_loss, color="tomato",
        linewidth=2, linestyle="--", label="Validation Loss")

ax.set_title("Training vs Validation Loss", fontsize=14)
ax.set_xlabel("Epoch")
ax.set_ylabel("Loss")
ax.legend()
ax.grid(True, linestyle="--", alpha=0.4)
plt.tight_layout()
plt.savefig("loss_curve.png", dpi=150, bbox_inches="tight")
plt.show()

Bar Chart — Comparing Categories

Use a bar chart when comparing discrete categories. In AI, bar charts are used to compare model performance across classes or datasets.

import matplotlib.pyplot as plt
import numpy as np

models  = ["ResNet50", "VGG16", "MobileNet", "EfficientNet"]
accuracy = [0.923,     0.901,   0.887,       0.941]

fig, ax = plt.subplots(figsize=(8, 5))

bars = ax.bar(models, accuracy,
              color=["steelblue", "tomato", "mediumseagreen", "mediumpurple"],
              width=0.5, edgecolor="white")

# Add value labels on top of each bar
for bar, val in zip(bars, accuracy):
    ax.text(bar.get_x() + bar.get_width() / 2,
            bar.get_height() + 0.005,
            f"{val:.1%}",
            ha="center", va="bottom", fontsize=10)

ax.set_title("Model Accuracy Comparison", fontsize=14)
ax.set_ylabel("Accuracy")
ax.set_ylim(0.85, 0.97)
ax.grid(True, axis="y", linestyle="--", alpha=0.4)
plt.tight_layout()
plt.savefig("model_accuracy.png", dpi=150, bbox_inches="tight")
plt.show()

Horizontal Bar Chart

Use a horizontal bar chart when category names are long or when ranking is the main message.

import matplotlib.pyplot as plt

features   = ["Age", "Income", "Education", "Experience", "Location"]
importance = [0.31,  0.28,     0.19,        0.14,         0.08]

# Sort by importance
sorted_pairs = sorted(zip(importance, features))
importance_s, features_s = zip(*sorted_pairs)

fig, ax = plt.subplots(figsize=(8, 4))
ax.barh(features_s, importance_s, color="steelblue", edgecolor="white")

ax.set_title("Feature Importance", fontsize=14)
ax.set_xlabel("Importance Score")
ax.grid(True, axis="x", linestyle="--", alpha=0.4)
plt.tight_layout()
plt.savefig("feature_importance.png", dpi=150, bbox_inches="tight")
plt.show()

Scatter Plot — Relationships Between Variables

Use a scatter plot to show how two continuous variables relate. In AI, scatter plots reveal correlations, clusters, and outliers in your data.

import matplotlib.pyplot as plt
import numpy as np

np.random.seed(42)

# Simulate two features and a binary class label
n = 100
class0_x = np.random.normal(2, 0.8, n)
class0_y = np.random.normal(2, 0.8, n)
class1_x = np.random.normal(4, 0.8, n)
class1_y = np.random.normal(4, 0.8, n)

fig, ax = plt.subplots(figsize=(7, 6))

ax.scatter(class0_x, class0_y, color="steelblue",
           alpha=0.6, label="Class 0", edgecolors="white", s=60)
ax.scatter(class1_x, class1_y, color="tomato",
           alpha=0.6, label="Class 1", edgecolors="white", s=60)

ax.set_title("Feature Space — Two Classes", fontsize=14)
ax.set_xlabel("Feature 1")
ax.set_ylabel("Feature 2")
ax.legend()
ax.grid(True, linestyle="--", alpha=0.3)
plt.tight_layout()
plt.savefig("scatter_classes.png", dpi=150, bbox_inches="tight")
plt.show()
💡 Use alpha (transparency) between 0.4 and 0.7 on scatter plots with many points — it prevents overplotting and lets you see where data is dense.

Histogram — Data Distribution

A histogram shows how values are distributed across a range. In AI, histograms help you spot skewed data, outliers, and whether your features need normalisation.

import matplotlib.pyplot as plt
import numpy as np

np.random.seed(0)

# Simulate model prediction confidence scores
scores = np.concatenate([
    np.random.normal(0.85, 0.08, 300),  # mostly correct predictions
    np.random.normal(0.40, 0.10, 100)   # uncertain predictions
])
scores = np.clip(scores, 0, 1)

fig, ax = plt.subplots(figsize=(8, 4))

ax.hist(scores, bins=30, color="steelblue",
        edgecolor="white", alpha=0.8)

# Add vertical line at threshold
ax.axvline(0.5, color="tomato", linestyle="--",
           linewidth=2, label="Decision threshold (0.5)")

ax.set_title("Distribution of Prediction Confidence", fontsize=14)
ax.set_xlabel("Confidence Score")
ax.set_ylabel("Count")
ax.legend()
ax.grid(True, axis="y", linestyle="--", alpha=0.4)
plt.tight_layout()
plt.savefig("confidence_distribution.png", dpi=150, bbox_inches="tight")
plt.show()

When to Use Each Chart Type

  • Line chart — data changes over time or a continuous sequence (loss curves, stock prices, sensor readings).
  • Bar chart — comparing a metric across a small number of discrete categories (model accuracy, class counts).
  • Horizontal bar — same as bar chart but when category labels are long or you are showing a ranking.
  • Scatter plot — relationship between two numerical variables (feature correlation, cluster visualisation).
  • Histogram — distribution of a single numerical variable (feature distribution, prediction scores).
⚠️ Never use a pie chart for more than 5 categories — bar charts communicate proportions more accurately. Avoid 3D charts entirely — they distort perception and add no information.
