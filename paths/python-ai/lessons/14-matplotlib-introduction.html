Matplotlib is Python's foundational data visualization library. Every chart, graph, and plot you see in AI research papers, data science notebooks, and technical reports is almost certainly created with Matplotlib or a library built on top of it. This lesson teaches you how Matplotlib is structured and how to produce your first clean, labelled charts from scratch.

Lesson Overview
  • Understand Matplotlib's two interfaces: pyplot and object-oriented.
  • Create a figure and axes using plt.subplots().
  • Add titles, axis labels, legends, and grid lines.
  • Save figures to PNG and PDF files.
  • Plot directly from a Pandas DataFrame.

Two Interfaces: pyplot vs Object-Oriented

Matplotlib has two ways to create charts. The pyplot interface is quick and script-like. The object-oriented (OO) interface gives you full control and is the professional standard for complex figures.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100)
y = np.sin(x)

# pyplot interface — quick and simple
plt.plot(x, y)
plt.title("Sine Wave")
plt.xlabel("x")
plt.ylabel("sin(x)")
plt.show()

# Object-oriented interface — recommended
fig, ax = plt.subplots()
ax.plot(x, y)
ax.set_title("Sine Wave")
ax.set_xlabel("x")
ax.set_ylabel("sin(x)")
plt.show()
💡 Always use the object-oriented interface (fig, ax = plt.subplots()) for any chart you will share or publish. It makes the code easier to read, debug, and extend — especially when you have multiple subplots.

Anatomy of a Matplotlib Figure

Understanding the structure of a Matplotlib figure prevents confusion when customising charts.

  • Figure — the entire canvas. Can contain one or more axes.
  • Axes — a single plot area with its own x and y axis, title, and labels. Not the same as "axis".
  • Axis — the x or y number line on an Axes object.
  • Artist — everything drawn on the figure: lines, text, ticks, markers.
import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)

fig, ax = plt.subplots(figsize=(8, 4))  # width=8, height=4 inches

ax.plot(x, np.sin(x), label="sin(x)")
ax.plot(x, np.cos(x), label="cos(x)")

# Labels and title
ax.set_title("Sine and Cosine", fontsize=14, fontweight="bold")
ax.set_xlabel("Angle (radians)")
ax.set_ylabel("Value")

# Legend
ax.legend()

# Grid
ax.grid(True, linestyle="--", alpha=0.5)

# Axis limits
ax.set_xlim(0, 2 * np.pi)
ax.set_ylim(-1.5, 1.5)

plt.tight_layout()
plt.show()

Saving Figures

Always save your figures as files when working on a project — do not rely on the interactive window. Use savefig() before show().

import matplotlib.pyplot as plt
import numpy as np

fig, ax = plt.subplots(figsize=(8, 4))
x = np.linspace(0, 10, 100)
ax.plot(x, np.sin(x))
ax.set_title("Sine Wave")

# Save before show()
fig.savefig("sine_wave.png", dpi=150, bbox_inches="tight")
fig.savefig("sine_wave.pdf", bbox_inches="tight")

plt.show()
print("Figure saved.")
ℹ️ dpi=150 sets resolution (dots per inch). Use 150 for screen, 300 for print. bbox_inches="tight" removes whitespace around the figure. Always use both when saving for a report or presentation.

Colors, Line Styles, and Markers

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 50)

fig, ax = plt.subplots(figsize=(9, 4))

# Color — name, hex, or RGB
ax.plot(x, np.sin(x), color="steelblue", label="sin")

# Line style: '-' solid, '--' dashed, ':' dotted, '-.' dash-dot
ax.plot(x, np.cos(x), color="#e74c3c", linestyle="--", label="cos")

# Line width
ax.plot(x, np.sin(x) * 0.5, linewidth=3, color="green", label="0.5*sin")

# Markers: 'o' circle, 's' square, '^' triangle, 'D' diamond
ax.plot(x[::5], np.sin(x[::5]), marker="o", linestyle="none",
        color="orange", markersize=8, label="sample points")

ax.legend()
ax.set_title("Line Styles, Colors and Markers")
ax.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

Plotting Directly from Pandas

Pandas DataFrames have a built-in .plot() method that calls Matplotlib under the hood. It is great for quick exploration.

import pandas as pd
import matplotlib.pyplot as plt

data = {
    "month":   ["Jan", "Feb", "Mar", "Apr", "May", "Jun"],
    "revenue": [1200, 1500, 1100, 1800, 2100, 1900],
    "costs":   [800,  900,  850,  1100, 1200, 1050]
}
df = pd.DataFrame(data)

# Line plot from DataFrame
ax = df.plot(x="month", y=["revenue", "costs"],
             kind="line", figsize=(8, 4),
             title="Revenue vs Costs")
ax.set_xlabel("Month")
ax.set_ylabel("Amount (USD)")
ax.grid(True, alpha=0.3)
plt.tight_layout()
plt.savefig("revenue_costs.png", dpi=150, bbox_inches="tight")
plt.show()
