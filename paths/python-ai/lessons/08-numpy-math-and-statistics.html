NumPy ships with a comprehensive library of mathematical and statistical functions that operate on entire arrays in one call. These functions are the building blocks of every AI algorithm — from computing loss functions and activation functions to normalising datasets and analysing model performance. This lesson covers the functions you will reach for every single day.

Lesson Overview
  • Use aggregate functions: sum, min, max, mean, std, var.
  • Apply functions along a specific axis of a 2D array.
  • Use universal functions (ufuncs): np.sqrt, np.exp, np.log, np.abs.
  • Perform matrix operations: dot product, matrix multiplication, transpose.
  • Sort arrays and find unique values.

Aggregate Functions

Aggregate functions reduce an array to a single value or a smaller array. Every AI metric — accuracy, loss, mean squared error — is an aggregate.

import numpy as np

scores = np.array([0.72, 0.85, 0.91, 0.68, 0.95, 0.78, 0.83])

print(np.sum(scores))    # 5.72
print(np.min(scores))    # 0.68
print(np.max(scores))    # 0.95
print(np.mean(scores))   # 0.817
print(np.median(scores)) # 0.83
print(np.std(scores))    # 0.089  — standard deviation
print(np.var(scores))    # 0.008  — variance

# Equivalent method syntax
print(scores.mean())
print(scores.std())
print(scores.max())

# argmin / argmax — index of the min/max value
print(np.argmin(scores))  # 3  (value 0.68)
print(np.argmax(scores))  # 4  (value 0.95)

Working Along an Axis

For 2D arrays you can apply aggregate functions along axis=0 (down columns) or axis=1 (across rows). This is essential when working with datasets where rows are samples and columns are features.

import numpy as np

# 4 students, 3 exam scores each
results = np.array([[85, 90, 78],
                    [92, 88, 95],
                    [70, 75, 80],
                    [88, 92, 85]])

# axis=0 — aggregate DOWN each column (per exam)
print(np.mean(results, axis=0))
# [83.75 86.25 84.5] — average score per exam

# axis=1 — aggregate ACROSS each row (per student)
print(np.mean(results, axis=1))
# [84.33 91.67 75.   88.33] — average score per student

print(np.max(results, axis=1))   # best score per student
print(np.sum(results, axis=0))   # total per exam
💡 A simple way to remember axes: axis=0 collapses rows (operates down), axis=1 collapses columns (operates across). In ML: axis=0 is usually "per feature", axis=1 is "per sample".

Universal Functions (ufuncs)

Ufuncs are NumPy functions that apply element-wise to every value in an array with no loop. They are the fastest way to apply mathematical transformations to your data.

import numpy as np

x = np.array([1.0, 4.0, 9.0, 16.0, 25.0])

print(np.sqrt(x))      # [1. 2. 3. 4. 5.]
print(np.square(x))    # [  1.  16.  81. 256. 625.]
print(np.abs(np.array([-3, -1, 0, 2, 4])))  # [3 1 0 2 4]

# Exponential and log — core to loss functions
logits = np.array([2.0, 1.0, 0.1])
print(np.exp(logits))   # [ 7.389  2.718  1.105]
print(np.log(x))        # [0.    1.386 2.197 2.773 3.219]
print(np.log2(x))       # [0.    2.    3.170 4.    4.644]

# Trigonometric — used in positional encodings, signal processing
angles = np.linspace(0, np.pi, 5)
print(np.sin(angles).round(2))  # [0.   0.71 1.   0.71 0.  ]
print(np.cos(angles).round(2))  # [ 1.   0.71  0.  -0.71 -1.]

Matrix Operations

Matrix operations are the heart of neural networks. Every layer of a neural network is a matrix multiplication followed by a function application.

import numpy as np

A = np.array([[1, 2],
              [3, 4]])

B = np.array([[5, 6],
              [7, 8]])

# Transpose — flip rows and columns
print(A.T)
# [[1 3]
#  [2 4]]

# Element-wise multiplication (NOT matrix multiply)
print(A * B)
# [[ 5 12]
#  [21 32]]

# Matrix multiplication — use @ operator or np.dot()
print(A @ B)
# [[19 22]
#  [43 50]]

print(np.dot(A, B))   # same result

# Dot product of two 1D vectors
v1 = np.array([1, 2, 3])
v2 = np.array([4, 5, 6])
print(np.dot(v1, v2))  # 32  (1*4 + 2*5 + 3*6)

# Determinant and inverse — used in linear regression
print(np.linalg.det(A))       # -2.0
print(np.linalg.inv(A))       # inverse of A
print(np.linalg.norm(v1))     # 3.742 — Euclidean length
ℹ️ The @ operator for matrix multiplication was introduced in Python 3.5. Always use @ for matrix multiplication and * only when you want element-wise multiplication.

Sorting and Unique Values

import numpy as np

scores = np.array([0.72, 0.91, 0.65, 0.85, 0.91, 0.78])

# Sort ascending (returns new array)
print(np.sort(scores))
# [0.65 0.72 0.78 0.85 0.91 0.91]

# argsort — indices that would sort the array
order = np.argsort(scores)
print(order)   # indices of ascending order

# Sort descending
print(np.sort(scores)[::-1])

# Unique values and their counts
labels = np.array([0, 1, 2, 1, 0, 2, 2, 1])
unique, counts = np.unique(labels, return_counts=True)
print(unique)  # [0 1 2]
print(counts)  # [2 3 3] — class distribution

Clipping and Rounding

Clipping and rounding are common preprocessing steps — clipping keeps values within a valid range, rounding controls precision.

import numpy as np

raw = np.array([-0.5, 0.2, 1.3, 0.8, -0.1, 1.0])

# clip — values below min become min, above max become max
clipped = np.clip(raw, 0.0, 1.0)
print(clipped)  # [0.  0.2 1.  0.8 0.  1. ]

# round
vals = np.array([0.1234, 0.5678, 0.9999])
print(np.round(vals, 2))   # [0.12 0.57 1.  ]
print(np.floor(vals))      # [0. 0. 0.]
print(np.ceil(vals))       # [1. 1. 1.]
