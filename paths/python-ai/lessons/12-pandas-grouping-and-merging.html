After loading and cleaning data, the next step is to extract meaningful insights. Grouping lets you split a dataset by a category and compute statistics for each group. Merging lets you combine multiple DataFrames into one — exactly like SQL JOIN operations. These are the two most powerful tools in a data scientist's Pandas toolkit.

Lesson Overview
  • Use groupby() to split data into groups and compute aggregations.
  • Use agg() to compute multiple statistics at once.
  • Use pivot_table() to reshape data into a summary table.
  • Merge DataFrames with merge() using inner, left, right, and outer joins.
  • Concatenate DataFrames vertically and horizontally with concat().

groupby — Split, Apply, Combine

groupby() follows the split-apply-combine pattern: split the data by a category, apply a function to each group, combine the results into a new DataFrame.

import pandas as pd

data = {
    "city":    ["Addis", "Nairobi", "Addis", "Lagos", "Nairobi", "Lagos", "Addis"],
    "product": ["A", "B", "B", "A", "A", "B", "A"],
    "sales":   [200, 350, 180, 420, 310, 290, 250],
    "units":   [10, 15, 8, 20, 14, 12, 11]
}
df = pd.DataFrame(data)

# Group by one column and compute mean
print(df.groupby("city")["sales"].mean())
# Addis      210.0
# Lagos      355.0
# Nairobi    330.0

# Group by one column, compute multiple stats
print(df.groupby("city")["sales"].agg(["mean", "sum", "max", "count"]))

# Group by multiple columns
print(df.groupby(["city", "product"])["sales"].sum())

Custom Aggregation with agg()

agg() lets you apply different functions to different columns in one step.

import pandas as pd

data = {
    "city":  ["Addis", "Nairobi", "Addis", "Lagos", "Nairobi", "Lagos"],
    "sales": [200, 350, 180, 420, 310, 290],
    "units": [10, 15, 8, 20, 14, 12]
}
df = pd.DataFrame(data)

# Different aggregations per column
result = df.groupby("city").agg(
    total_sales=("sales", "sum"),
    avg_sales=("sales", "mean"),
    max_units=("units", "max"),
    num_records=("units", "count")
)
print(result)
#          total_sales  avg_sales  max_units  num_records
# Addis            380      190.0         10            2
# Lagos            710      355.0         20            2
# Nairobi          660      330.0         15            2

Pivot Tables

A pivot table reshapes data into a summary matrix — rows are one category, columns are another, values are an aggregation. This is the same as Excel's PivotTable feature.

import pandas as pd

data = {
    "city":    ["Addis", "Nairobi", "Addis", "Lagos", "Nairobi", "Lagos", "Addis"],
    "product": ["A", "B", "B", "A", "A", "B", "A"],
    "sales":   [200, 350, 180, 420, 310, 290, 250]
}
df = pd.DataFrame(data)

pivot = pd.pivot_table(
    df,
    values="sales",       # what to aggregate
    index="city",         # rows
    columns="product",    # columns
    aggfunc="sum",        # how to aggregate
    fill_value=0          # replace NaN with 0
)
print(pivot)
# product    A    B
# city
# Addis    450  180
# Lagos    420  290
# Nairobi  310  350

Merging DataFrames

pd.merge() combines two DataFrames on a common column — exactly like SQL JOIN. The how parameter controls which rows to keep.

import pandas as pd

students = pd.DataFrame({
    "student_id": [1, 2, 3, 4],
    "name":       ["Alice", "Bob", "Carol", "David"]
})

scores = pd.DataFrame({
    "student_id": [1, 2, 3, 5],
    "score":      [0.85, 0.92, 0.78, 0.95]
})

# Inner join — only rows that exist in BOTH
inner = pd.merge(students, scores, on="student_id", how="inner")
print(inner)
# student_id   name  score
#          1  Alice   0.85
#          2    Bob   0.92
#          3  Carol   0.78

# Left join — all rows from left, match from right
left = pd.merge(students, scores, on="student_id", how="left")
print(left)
# student_id   name  score
#          1  Alice   0.85
#          2    Bob   0.92
#          3  Carol   0.78
#          4  David    NaN  ← David has no score

# Outer join — all rows from both
outer = pd.merge(students, scores, on="student_id", how="outer")
print(outer)
💡 Use inner when you only want records that exist in both datasets. Use left when you want to keep all records from the primary dataset and add information from a secondary one where available.

Concatenating DataFrames

pd.concat() stacks DataFrames vertically (more rows) or horizontally (more columns). Use it when combining data from multiple files or sources with the same structure.

import pandas as pd

batch1 = pd.DataFrame({
    "name":  ["Alice", "Bob"],
    "score": [0.85, 0.92]
})

batch2 = pd.DataFrame({
    "name":  ["Carol", "David", "Eve"],
    "score": [0.78, 0.95, 0.88]
})

# Vertical stack — axis=0 (default)
combined = pd.concat([batch1, batch2], ignore_index=True)
print(combined)
#     name  score
# 0  Alice   0.85
# 1    Bob   0.92
# 2  Carol   0.78
# 3  David   0.95
# 4    Eve   0.88

# Horizontal stack — axis=1
extra = pd.DataFrame({"grade": ["B", "A", "C", "A", "B"]})
full = pd.concat([combined, extra], axis=1)
print(full)

Sorting and Ranking

import pandas as pd

data = {
    "name":  ["Alice", "Bob", "Carol", "David", "Eve"],
    "score": [0.85, 0.92, 0.78, 0.95, 0.88],
    "age":   [24, 31, 28, 35, 22]
}
df = pd.DataFrame(data)

# Sort by one column
print(df.sort_values("score", ascending=False))

# Sort by multiple columns
print(df.sort_values(["score", "age"], ascending=[False, True]))

# Rank — assign rank to each value
df["rank"] = df["score"].rank(ascending=False).astype(int)
print(df.sort_values("rank"))
