Congratulations on reaching the capstone. This is your final project for the Python for AI path — a complete, end-to-end data analysis and visualisation report that combines everything you have learned across all four units. You will load real data, clean it with Pandas, analyse it with NumPy, and communicate your findings with a professional multi-panel Matplotlib figure. This is what real data science work looks like.

Project Overview
  • Load and clean a real dataset using Pandas.
  • Perform numerical analysis using NumPy.
  • Answer five analytical questions with code and printed findings.
  • Produce a professional 6-panel visualisation figure saved as a PNG.
  • Organise everything into clean, modular Python files.

The Dataset

Use the World Happiness Report dataset. Download it and save as happiness.csv.

ℹ️ Download from: World Happiness Report 2023 — CSV. Save as happiness.csv in your project folder. Key columns: Country name, Ladder score (happiness score), Logged GDP per capita, Social support, Healthy life expectancy, Freedom to make life choices, Generosity, Perceptions of corruption, Regional indicator.

Requirements

  • Code split into two files: analysis.py (data loading, cleaning, analysis) and visualise.py (all charts).
  • main.py imports and runs both modules in sequence.
  • All functions must have docstrings.
  • Use plt.style.use("seaborn-v0_8-whitegrid") for all charts.
  • Final figure saved as happiness_report.png at dpi=150.
  • Script runs end-to-end with python3 main.py.

Five Analytical Questions

Your analysis.py must answer all five questions with printed output:

  1. Which are the top 10 happiest countries and what are their scores?
  2. Which region has the highest average happiness score?
  3. What is the correlation between GDP per capita and happiness score?
  4. Which factor (GDP, social support, health, freedom, generosity, corruption) correlates most strongly with happiness?
  5. How does average happiness differ between the top and bottom 25% of countries by GDP?

Six Required Charts

Your visualise.py must produce a single 3×2 figure containing all six charts:

  1. Top 15 countries — horizontal bar chart of happiness scores, colour-coded by region.
  2. Regional averages — bar chart of mean happiness score per region, sorted descending.
  3. GDP vs Happiness — scatter plot with a trend line, coloured by region.
  4. Distribution of happiness scores — histogram with mean and median lines annotated.
  5. Factor correlations — horizontal bar chart showing correlation of each factor with happiness score.
  6. Top vs Bottom GDP quartile — side-by-side bar chart comparing mean happiness.

Hints

  • Correlation: np.corrcoef(df["col1"], df["col2"])[0, 1] returns the Pearson r.
  • Trend line: coeffs = np.polyfit(x, y, 1) then trend = np.poly1d(coeffs)(x).
  • GDP quartiles: q25 = df["gdp"].quantile(0.25) and q75 = df["gdp"].quantile(0.75).
  • Colour by region: use pd.Categorical(df["region"]).codes as a colour array in scatter.
  • Rotating labels: ax.tick_params(axis="x", rotation=45).
  • Figure spacing: call plt.tight_layout() before savefig() to prevent label clipping.
💡 Build one chart at a time. Get each panel working individually before assembling them into the 3×2 figure. Use fig, ax = plt.subplots() for each while developing, then move the code into build_report() once it works.
