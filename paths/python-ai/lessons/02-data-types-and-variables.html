Every value in a Python program has a type. Whether you are storing a temperature reading, a patient name, or a list of model predictions, Python's built-in data types give you the right tool for the job. Understanding types and how to manipulate them with operators is the foundation of all AI data pipelines.

Lesson Overview
  • Work with Python's core types: int, float, str, bool, list, tuple, dict, and set.
  • Declare variables using Python's snake_case convention.
  • Use arithmetic, comparison, logical, and membership operators.
  • Convert between types with type casting.
  • Format strings using f-strings.

Variables

A variable is a name that refers to a value. Python uses dynamic typing — you don't declare the type, Python infers it. Variable names should be lowercase with underscores (snake_case), descriptive, and never start with a number.

learning_rate = 0.001
num_epochs = 50
model_name = "ResNet50"
is_trained = False

print(type(learning_rate))   # <class 'float'>
print(type(num_epochs))      # <class 'int'>
print(type(model_name))      # <class 'str'>
print(type(is_trained))      # <class 'bool'>

Numeric Types: int and float

Integers (int) are whole numbers. Floats (float) are decimal numbers. In AI, floats are everywhere — model weights, loss values, probabilities, and pixel intensities are all floats.

samples = 1000          # int
accuracy = 0.9423       # float
loss = 1.4e-3           # scientific notation = 0.0014

print(10 + 3)    # 13   — addition
print(10 - 3)    # 7    — subtraction
print(10 * 3)    # 30   — multiplication
print(10 / 3)    # 3.333 — true division (always float)
print(10 // 3)   # 3    — floor division
print(10 % 3)    # 1    — modulo (remainder)
print(10 ** 3)   # 1000 — exponentiation

print(abs(-5))            # 5
print(round(3.14159, 2))  # 3.14
⚠️ Floating-point arithmetic is not always exact. 0.1 + 0.2 gives 0.30000000000000004 due to IEEE 754 binary representation. Use round() when displaying results.

Strings

Strings (str) store text. In AI you use them for dataset paths, column names, labels, and log messages. Python strings are immutable — operations always return a new string.

dataset_path = "/data/train/images"
label = "cat"

# f-strings — the modern way to format strings
epoch = 5
acc = 0.9423
print(f"Epoch {epoch}: accuracy = {acc:.2f}")
# Epoch 5: accuracy = 0.94

print(label.upper())           # 'CAT'
print(label.capitalize())      # 'Cat'
print("  hello  ".strip())     # 'hello'
print("a,b,c".split(","))      # ['a', 'b', 'c']
print("hello".replace("l","r"))# 'herro'
print(len("neural network"))   # 14

# Slicing
path = "/data/train/images"
print(path[0])     # '/'
print(path[-6:])   # 'images'
print(path[1:5])   # 'data'

Lists, Tuples, Dicts, and Sets

Python's collection types let you group multiple values. Each has different characteristics that matter when working with data.

# List — ordered, mutable, allows duplicates
scores = [0.92, 0.87, 0.95, 0.91]
scores.append(0.98)
scores[0] = 0.93
print(scores[1])    # 0.87
print(len(scores))  # 5

# Tuple — ordered, immutable
image_size = (224, 224, 3)
print(image_size[2])  # 3

# Dictionary — key-value pairs
model_config = {
    "architecture": "ResNet50",
    "learning_rate": 0.001,
    "epochs": 100,
    "batch_size": 32
}
print(model_config["learning_rate"])  # 0.001
model_config["dropout"] = 0.5
print(list(model_config.keys()))

# Set — unordered, unique values only
labels = {"cat", "dog", "cat", "bird"}
print(labels)  # {'cat', 'dog', 'bird'}

Comparison and Logical Operators

Comparison operators return a boolean (True or False). Logical operators combine boolean expressions. These are essential in conditionals and data filtering.

accuracy = 0.95
threshold = 0.90

print(accuracy > threshold)    # True
print(accuracy == threshold)   # False
print(accuracy != threshold)   # True
print(accuracy >= 0.95)        # True

is_overfitting = True
val_accuracy = 0.80

print(accuracy > threshold and not is_overfitting)  # False
print(accuracy > threshold or val_accuracy > 0.75)  # True

classes = ["cat", "dog", "bird"]
print("cat" in classes)       # True
print("fish" not in classes)  # True

Type Casting

You can convert between types using constructor functions. This is common when reading data from files or user input.

print(int(3.9))       # 3   — truncates, does not round
print(float(5))       # 5.0
print(str(100))       # '100'
print(bool(0))        # False
print(bool(1))        # True
print(bool(""))       # False — empty string is falsy
print(bool("hello"))  # True

# Reading input always gives a string
age_input = "28"
age = int(age_input)
print(age + 2)        # 30
