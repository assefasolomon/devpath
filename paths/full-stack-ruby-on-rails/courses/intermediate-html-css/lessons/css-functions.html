CSS has dozens of built-in functions. This lesson covers the most useful ones: calc(), clamp(), min(), max(), var(), and the transform functions.

Lesson Overview
  • Use calc() for dynamic sizing.
  • Use clamp(), min(), max() for responsive values.
  • Use CSS custom properties with var().
  • Use transform functions for movement and scaling.

calc() and Fluid Sizing

/* calc — mix units */
.sidebar { width: calc(100% - 280px); }
.card    { padding: calc(1rem + 2vw); }

/* min() — smallest of the values */
.container { width: min(90%, 1200px); }   /* max-width: 1200px but never > 90% */

/* max() — largest of the values */
font-size: max(1rem, 2.5vw);   /* never smaller than 1rem */

/* clamp(min, preferred, max) */
h1 { font-size: clamp(1.5rem, 5vw, 3rem); }
.content { width: clamp(300px, 80%, 800px); }

Transform Functions

/* translate — move without affecting layout */
.modal { transform: translateX(-50%) translateY(-50%); }

/* Center an absolute element */
.centered {
  position: absolute;
  top: 50%; left: 50%;
  transform: translate(-50%, -50%);
}

/* scale, rotate */
.icon:hover { transform: scale(1.1) rotate(5deg); }

/* Combined with transition */
.btn { transition: transform 150ms ease; }
.btn:hover { transform: translateY(-2px); }
