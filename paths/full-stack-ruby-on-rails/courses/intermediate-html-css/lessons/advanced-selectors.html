CSS selectors go far beyond classes and IDs. Advanced selectors let you target elements by relationship, state, and content — keeping your HTML clean and your CSS precise.

Lesson Overview
  • Use child, sibling, and descendant combinators.
  • Use :is(), :where(), :not(), :has().
  • Use attribute selectors.
  • Use pseudo-elements ::before and ::after.

Combinators

/* Descendant — any depth */       .parent .child { }
/* Child — direct only */          .parent > .child { }
/* Adjacent sibling */             h2 + p { margin-top: 0; }
/* General sibling */              h2 ~ p { color: gray; }

Pseudo-classes and Pseudo-elements

/* Structural pseudo-classes */
li:first-child { font-weight: bold; }
li:last-child  { border-bottom: none; }
li:nth-child(odd)  { background: #f9fafb; }
li:nth-child(3n+1) { color: red; }

/* :is() — matches any of the list */
:is(h1, h2, h3) { line-height: 1.2; }

/* :not() */
button:not([disabled]) { cursor: pointer; }

/* :has() — parent selector (modern browsers) */
.card:has(img) { padding: 0; }

/* Attribute selectors */
a[href^="https"] { color: green; }   /* starts with */
a[href$=".pdf"]  { color: red; }     /* ends with */
input[type="email"] { border: blue; }

/* Pseudo-elements */
p::first-line { font-weight: bold; }
.required::after { content: " *"; color: red; }
