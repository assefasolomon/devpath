Emmet is a plugin built into VS Code that lets you write HTML and CSS with short abbreviations that expand into full markup. It dramatically speeds up writing templates.

Lesson Overview
  • Use Emmet abbreviations to scaffold HTML quickly.
  • Use CSS Emmet shortcuts.
  • Know the most useful Emmet patterns for Rails ERB files.

HTML Abbreviations

# Type these and press Tab in VS Code:

ul>li*5                  # <ul> with 5 <li> items
nav>ul>li*3>a[href=#]    # nav > ul > 3 links
.container>.card*3       # div.container with 3 div.card children
h1{Hello World}          # <h1>Hello World</h1>
input[type=email]#email  # <input type="email" id="email">
form>input*3+button{Submit}

# Numbering with $
ul>li.item-$*5
# => <li class="item-1"> ... <li class="item-5">

CSS Abbreviations

# In a CSS file:
m10          # margin: 10px
p10-20       # padding: 10px 20px
df           # display: flex
aic          # align-items: center
jcc          # justify-content: center
bgc#fff      # background-color: #fff
fw700        # font-weight: 700
