Browsers apply default CSS to every HTML element — margins, padding, font sizes, and more. Understanding these defaults and how to reset or normalise them is essential for consistent cross-browser styling.

Lesson Overview
  • Understand browser default (user-agent) styles.
  • Know the difference between a CSS reset and a normalise stylesheet.
  • Use box-sizing: border-box universally.

CSS Resets and Normalise

/* Minimal modern reset — include at the top of your stylesheet */
*, *::before, *::after { box-sizing: border-box; }
* { margin: 0; padding: 0; }
body { line-height: 1.5; -webkit-font-smoothing: antialiased; }
img, video, svg { display: block; max-width: 100%; }
input, button, textarea, select { font: inherit; }
p, h1, h2, h3, h4, h5, h6 { overflow-wrap: break-word; }
💡

Tailwind CSS includes a reset (Preflight) automatically. If you use Tailwind in Rails you get this for free.
