HTML tables are for tabular data — rows and columns of related information. They are often misused for layout (don't do that), but are the right tool for data grids, pricing tables, and comparison pages.

Lesson Overview
  • Build a complete HTML table with thead, tbody, tfoot.
  • Use colspan and rowspan.
  • Style tables with CSS.
  • Know when to use a table vs a list.

Table Structure

<table>
  <thead>
    <tr>
      <th scope="col">Name</th>
      <th scope="col">Role</th>
      <th scope="col">Joined</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Alice</td>
      <td>Admin</td>
      <td>Jan 2024</td>
    </tr>
    <tr>
      <td>Bob</td>
      <td>Member</td>
      <td>Feb 2024</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td colspan="3">2 users total</td>
    </tr>
  </tfoot>
</table>

Styling Tables

table { width: 100%; border-collapse: collapse; }
th, td { padding: 12px 16px; text-align: left; border-bottom: 1px solid #e5e7eb; }
thead th { background: #f9fafb; font-weight: 600; }
tbody tr:hover { background: #f0f9ff; }
