CSS features have different levels of support across browsers and versions. This lesson covers how to check compatibility, use fallbacks, and progressively enhance your stylesheets.

Lesson Overview
  • Use caniuse.com to check feature support.
  • Write fallbacks for unsupported CSS.
  • Use @supports for feature queries.
  • Understand vendor prefixes and autoprefixer.

@supports

/* @supports — apply styles only if browser supports a property */
@supports (display: grid) {
  .layout { display: grid; grid-template-columns: repeat(3, 1fr); }
}

/* Fallback for non-supporting browsers */
.layout { display: flex; flex-wrap: wrap; }  /* fallback */
@supports (display: grid) {
  .layout { display: grid; }                 /* enhancement */
}

/* Check for :has() support */
@supports selector(:has(a, b)) {
  .card:has(img) { padding: 0; }
}
