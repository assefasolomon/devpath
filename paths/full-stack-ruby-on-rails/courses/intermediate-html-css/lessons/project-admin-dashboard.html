Project: Admin Dashboard

Build a responsive admin dashboard using HTML and CSS only. This project exercises positioning, grid, custom properties, advanced selectors, and everything else from this course.

Lesson Overview
  • Build a fixed sidebar navigation.
  • Create a responsive grid of stat cards.
  • Build a data table with hover styles.
  • Apply a consistent design system using custom properties.

Requirements

  • Fixed sidebar with navigation links and a logo
  • Top header bar with search input and user avatar
  • Grid of 4 stat cards (Total Users, Revenue, Orders, Growth)
  • Recent activity table with 5 rows
  • Responsive: sidebar collapses on mobile
  • Dark mode via prefers-color-scheme and CSS custom properties
mkdir ~/devpath-projects/intermediate-html-css/admin-dashboard
cd ~/devpath-projects/intermediate-html-css/admin-dashboard
touch index.html styles.css

# Recommended structure:
# index.html
# styles/

#   variables.css
#   layout.css
#   components.css
💡

Design the layout in CSS Grid first (sidebar | main), then fill in the components. Use custom properties for all colors before writing a single element style.
