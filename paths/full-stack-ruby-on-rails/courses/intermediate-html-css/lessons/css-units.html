CSS has many units for size and spacing. Choosing the right unit — px, em, rem, %, vw, vh, ch — determines how your layout responds to font size changes, screen size, and user preferences.

Lesson Overview
  • Know when to use px, em, rem, %, vw/vh, and ch.
  • Understand how em and rem cascade.
  • Use relative units for accessible, responsive designs.

Unit Reference

/* px — absolute, ignores user font preferences */
font-size: 16px;

/* rem — relative to root (html) font size — USE FOR FONT SIZES */
font-size: 1rem;     /* = 16px if root is 16px */
font-size: 1.25rem;  /* = 20px */

/* em — relative to PARENT element font size — USE FOR SPACING */
padding: 1em;        /* = 1× the element's own font-size */
margin-bottom: 0.5em;

/* % — relative to parent dimension */
width: 50%;          /* half of parent width */

/* vw / vh — viewport width / height */
height: 100vh;       /* full screen height */
width: 100vw;

/* ch — width of the "0" character — great for line length */
max-width: 65ch;     /* readable paragraph width */

/* clamp(min, preferred, max) — fluid sizing */
font-size: clamp(1rem, 2.5vw, 1.5rem);
