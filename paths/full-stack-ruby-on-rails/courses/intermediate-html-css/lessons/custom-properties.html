CSS custom properties (variables) let you define values once and reuse them everywhere. They are the foundation of theming, dark mode, and design systems.

Lesson Overview
  • Define and use CSS custom properties.
  • Scope variables to components.
  • Build a simple theme system with custom properties.
  • Implement dark mode with prefers-color-scheme.

Custom Properties

/* Define on :root — available everywhere */
:root {
  --color-primary:    #3b82f6;
  --color-secondary:  #10b981;
  --color-text:       #111827;
  --color-bg:         #ffffff;
  --spacing-sm:       0.5rem;
  --spacing-md:       1rem;
  --spacing-lg:       2rem;
  --radius:           8px;
  --shadow:           0 1px 3px rgba(0,0,0,.1);
  --font-sans:        'Inter', system-ui, sans-serif;
}

/* Use with var() */
.btn {
  background: var(--color-primary);
  padding: var(--spacing-sm) var(--spacing-md);
  border-radius: var(--radius);
  box-shadow: var(--shadow);
}

/* Fallback value */
color: var(--color-brand, #3b82f6);

Dark Mode

:root {
  --color-bg:   #ffffff;
  --color-text: #111827;
}

@media (prefers-color-scheme: dark) {
  :root {
    --color-bg:   #111827;
    --color-text: #f9fafb;
  }
}

body { background: var(--color-bg); color: var(--color-text); }
/* All components automatically switch — no extra CSS needed */
