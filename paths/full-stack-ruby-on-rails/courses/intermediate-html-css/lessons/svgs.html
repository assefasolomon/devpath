SVGs (Scalable Vector Graphics) are XML-based images that scale to any size without losing quality. They are used for icons, logos, and illustrations throughout modern web applications.

Lesson Overview
  • Understand SVG syntax and coordinate system.
  • Use SVG inline and as an img src.
  • Style SVGs with CSS.
  • Use SVG icons from a sprite or icon library.

SVG Syntax

<!-- Inline SVG — can be styled with CSS -->
<svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg" width="50" height="50">
  <circle cx="50" cy="50" r="40" fill="currentColor" />
  <rect x="20" y="20" width="30" height="30" fill="blue" />
  <path d="M10 80 L50 10 L90 80 Z" fill="red" />
  <text x="50" y="50" text-anchor="middle" font-size="12">SVG</text>
</svg>

Styling SVGs

/* Inline SVGs inherit currentColor */
.icon { color: #3b82f6; }
.icon svg { width: 24px; height: 24px; fill: currentColor; }

/* Target SVG elements directly */
svg circle { fill: tomato; }
svg:hover path { stroke: navy; }

SVG in Rails Views

<!-- As img tag — cannot be CSS-styled -->
<img src="/assets/logo.svg" alt="Logo" width="120">

<!-- Inline — full CSS/JS access — use a partial -->
<!-- app/views/shared/_icon_check.html.erb -->
<svg viewBox="0 0 24 24" fill="none" stroke="currentColor" stroke-width="2">
  <polyline points="20 6 9 17 4 12"/>
</svg>

<!-- Use it anywhere -->
<%= render "shared/icon_check" %>
