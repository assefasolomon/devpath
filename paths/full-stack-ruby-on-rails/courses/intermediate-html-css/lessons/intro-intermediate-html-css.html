This course bridges the gap between basic HTML/CSS and the advanced techniques used in professional web development. You will cover intermediate HTML elements and deeper CSS concepts.

Lesson Overview
  • Know what this course covers.
  • Understand how intermediate HTML/CSS fits into the Rails stack.

Course Overview

In the Foundations course you learned the building blocks. Here you go deeper: semantic HTML, SVGs, tables, advanced CSS selectors, custom properties, and modern layout techniques. These skills are used every time you write ERB views in Rails.
