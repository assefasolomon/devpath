This lesson covers CSS properties that come up constantly in real projects but aren't covered in introductory courses: overflow, opacity, cursor, border-radius, box-shadow, and object-fit.

Lesson Overview
  • Control overflow with overflow, overflow-x, overflow-y.
  • Use opacity and visibility correctly.
  • Apply box-shadow and drop-shadow.
  • Use object-fit for images.

Overflow

.container { overflow: hidden; }   /* clip content */
.scroll    { overflow: auto; }     /* scroll only when needed */
.x-scroll  { overflow-x: auto; overflow-y: hidden; }

/* Overflow and border-radius — clip child images */
.card { border-radius: 8px; overflow: hidden; }

Visual Properties

/* opacity vs visibility */
.hidden   { opacity: 0; }          /* invisible but takes up space, can be transitioned */
.gone     { visibility: hidden; }  /* invisible, takes up space, no transition */
.removed  { display: none; }       /* removed from layout */

/* box-shadow */
.card { box-shadow: 0 1px 3px rgba(0,0,0,.1), 0 1px 2px rgba(0,0,0,.06); }
.card:hover { box-shadow: 0 10px 25px rgba(0,0,0,.15); }

/* object-fit — for img inside a fixed container */
.avatar { width: 64px; height: 64px; border-radius: 50%; object-fit: cover; }
.hero-img { width: 100%; height: 400px; object-fit: cover; object-position: top; }
