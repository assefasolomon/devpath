PostgreSQL is the recommended database for Rails applications in production. This lesson covers installation, configuration, and switching from SQLite to PostgreSQL.

Lesson Overview
  • Install PostgreSQL on your operating system.
  • Configure database.yml for PostgreSQL.
  • Create development and test databases.
  • Use Rails with PostgreSQL-specific features.

Installation

# macOS (Homebrew)
brew install postgresql@16
brew services start postgresql@16

# Ubuntu / WSL
sudo apt install postgresql postgresql-contrib
sudo service postgresql start

# Create a PostgreSQL user
sudo -u postgres createuser --superuser $USER

# Verify
psql --version
psql -c "SELECT version();"

# config/database.yml
default: &default
  adapter: postgresql
  encoding: unicode
  pool: <%= ENV.fetch("RAILS_MAX_THREADS") { 5 } %>

development:
  <<: *default
  database: myapp_development
  username: <%= ENV['DB_USERNAME'] %>
  password: <%= ENV['DB_PASSWORD'] %>

test:
  <<: *default
  database: myapp_test

production:
  <<: *default
  url: <%= ENV['DATABASE_URL'] %>

# Create databases
rails db:create

# Or using psql directly
createdb myapp_development
createdb myapp_test
