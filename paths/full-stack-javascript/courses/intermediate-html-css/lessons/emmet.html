Emmet is a plugin built into VS Code that expands short abbreviations into full HTML and CSS. Once you learn it, writing HTML by hand feels painfully slow by comparison.

Lesson Overview
  • Use Emmet abbreviations to generate HTML structure instantly.
  • Chain, nest, and multiply elements with Emmet syntax.
  • Use Emmet for CSS property shortcuts.

HTML Abbreviations

Type an abbreviation in a .html file and press Tab:

! + Tab
→ Full HTML5 boilerplate

div.container + Tab
→ <div class="container"></div>

div#hero + Tab
→ <div id="hero"></div>

div.card.featured + Tab
→ <div class="card featured"></div>

Nesting and Multiplying

ul>li*5 + Tab
→ <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
  </ul>

nav>ul>li*3>a + Tab
→ <nav>
    <ul>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
      <li><a href=""></a></li>
    </ul>
  </nav>

div.card*3 + Tab
→ three div.card elements side by side

Content and Attributes

h1{Hello World} + Tab
→ <h1>Hello World</h1>

a[href=https://example.com]{Visit} + Tab
→ <a href="https://example.com">Visit</a>

input[type=email placeholder="Enter email"] + Tab
→ <input type="email" placeholder="Enter email">

p.item$*3 + Tab
→ <p class="item1"></p>
  <p class="item2"></p>
  <p class="item3"></p>

Siblings and Climbing

/* + creates a sibling, ^ climbs up one level */
div>p+span + Tab
→ <div>
    <p></p>
    <span></span>
  </div>

header>nav^main>section + Tab
→ <header><nav></nav></header>
  <main><section></section></main>
💡

You do not need to memorise all of this now. Start with the basic element, class, ID, and multiplication syntax. Add more as you naturally want to type faster.

  2. Build the skeleton of a blog post page (header, nav, main with two article elements each containing h2 and p, sidebar, footer) using Emmet in one line.
