This lesson revisits Natural Responsiveness with a focus on advanced techniques — container queries, fluid type systems, and performance optimisation for images.

Lesson Overview
  • Apply advanced Natural Responsiveness techniques to complex layouts.
  • Use container queries for component-level responsiveness.
  • Optimise images for Core Web Vitals.

Advanced Techniques

You covered the fundamentals in Intermediate HTML and CSS. This lesson focuses on the advanced patterns used in production applications.

/* Container queries — component responds to its own container, not viewport */
.card-container {
  container-type: inline-size;
}

@container (min-width: 400px) {
  .card {
    display: grid;
    grid-template-columns: 200px 1fr;
  }
}

/* Fluid type scale — no media queries needed */
:root {
  --step-0: clamp(1rem,     1vw + 0.75rem,  1.125rem);
  --step-1: clamp(1.25rem,  1.5vw + 0.875rem, 1.5rem);
  --step-2: clamp(1.563rem, 2vw + 1rem,     2rem);
  --step-3: clamp(1.953rem, 2.5vw + 1.25rem, 2.5rem);
  --step-4: clamp(2.441rem, 3vw + 1.5rem,   3.5rem);
}

p  { font-size: var(--step-0); }
h3 { font-size: var(--step-2); }
h1 { font-size: var(--step-4); }
