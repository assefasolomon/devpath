The Advanced HTML and CSS capstone. You will build a polished, fully accessible, animated, and responsive personal homepage — demonstrating every technique from this course.

Lesson Overview
  • Build a complete animated responsive homepage.
  • Apply CSS transforms, transitions, and keyframe animations.
  • Achieve 95+ on Lighthouse accessibility audit.
  • Use container queries and a fluid type scale.

Requirements

  • All requirements from the Intermediate HTML and CSS Homepage project, plus:
  • Entrance animations using keyframes and @keyframes
  • Smooth hover transitions on all interactive elements
  • At least one 3D CSS transform effect
  • Container queries on at least one component
  • Fluid typography using clamp()
  • prefers-reduced-motion media query disabling all animations
  • Lighthouse Accessibility score of 95+
  • Lighthouse Performance score of 90+
